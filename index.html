Data Flow & Access Controls

Secure, transparent, real-time data architecture built for clinical trust

Device Capture

Arduino reads sensors (pulse sensor, DHT22, LCD display) and forwards packets to ESP8266 NodeMCU

Secure Transmission

NodeMCU authenticates to ThingSpeak and pushes HBM, TEMPERATURE, and HUMIDITY to Channel 3128115

Web Portal Sync

HealthSync reads ThingSpeak API to display live charts, downloadable reports, and alerts

Device Setup & How It Works

Connect compact IoT hardware to secure cloud for real-time clinical monitoring

System Overview

HealthSync connects Arduino Uno with sensors to ESP8266 NodeMCU for cloud relay. Data flows to ThingSpeak channel 3128115, then to our dashboard for live charts, PDF reports, and remote consultations.

What You'll Need

  • Arduino Uno
  • ESP8266 NodeMCU
  • Pulse sensor (PPG)
  • DHT22 temperature & humidity sensor
  • 16x2 or 20x4 LCD display
  • Breadboard and jumper wires
  • USB cable and power supply
  • Stable Wi‑Fi network
ThingSpeak Channel: 3128115
