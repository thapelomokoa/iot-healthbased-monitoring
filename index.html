About this Demo

This product-demo shows how Arduino/NodeMCU sensors publish BPM, temperature, and humidity to ThingSpeak. The front-end fetches the read API to show live graphs and gauges.

  • ThingSpeak live pull (no writes from this UI)
  • CSV download of recent feeds
  • Responsive UI for product demos and presentations
