This is an **IoT-based Health Monitoring System** that uses:
- Arduino Uno  
- ESP8266 NodeMCU  
- Pulse Sensor  
- DHT22 Sensor  
- LCD Display  

The system collects **Heart Beat (BPM)**, **Temperature**, and **Humidity** data and sends it to **ThingSpeak Cloud** for real-time monitoring.  
Doctors and gym trainers can view live data dashboards, download reports as PDF, and consult patients remotely.
